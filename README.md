# Setup & Running the Script

## Bookmarklet
Create a bookmark, edit it, and then paste this into the URL address field:
```js
javascript: fetch("https://raw.githubusercontent.com/dawoodhq/grid-image-overlay/refs/heads/main/min.index.js").then(t=>t.text()).then(eval);
```
Use this bookmark on any page to load the script with one click.
